# Memory Game

This is a classic Memory Game that can be simply included in any html page like this:

    <html>
      <head>
        <script src="unpkg">link to unpkg</script>
      </head>
      <body>
        <div class="container">
              <mini-memory></mini-memory>
        </div>
      </body>
    </html>

** Please note Internet Explorer 11 is not supported **
